# CV Final Test Review Materials

这个仓库整理的是一套基于课程 PDF 和课堂重点线索生成的计算机视觉期末复习材料。

## 主要内容

- [emphasis/emphasis_detailed_review.md](emphasis/emphasis_detailed_review.md)
  - 逐条划重点详解版
  - 每个重点尽量配课件原话、原 PPT 图、课内题目、小白解释和考试答法
  - 第 31 章是高频考点精简强化版，适合临考快速背公式、流程和答题模板

- [emphasis/in_class_exercises_analysis.md](emphasis/in_class_exercises_analysis.md)
  - 课内出现过的题目专题分析
  - 按原课件题目页整理考点、答案和可能变体

- [emphasis/beginner_review.md](emphasis/beginner_review.md)
  - 面向零基础的小白展开讲解版
  - 重点会解释“这是什么、为什么会这样、考试怎么答”
  - 并尽量标出原课件位置和对应的 PPT 原图

- [emphasis/final_review.md](emphasis/final_review.md)
  - 更浓缩的提纲式复习版
  - 适合后期快速收束

- [emphasis/question_bank.md](emphasis/question_bank.md)
  - 覆盖 `emphasis` 划重点的题库
  - 含答案与解析

## 素材目录

- `cv_pdf/`
  - 原始课程 PDF

- `emphasis/ppt_originals/`
  - 从原课件中截取出来、用于贴在相应知识点旁边的原 PPT 图片

- `emphasis/generated_images/`
  - 为帮助理解难点而额外绘制的示意图

- `emphasis/github_images/`
  - 为了 GitHub 稳定显示而整理出的英文文件名课件页面图

## 当前建议阅读顺序

1. 先看 `emphasis_detailed_review.md`
2. 再看 `in_class_exercises_analysis.md`
3. 然后刷 `question_bank.md`
4. 最后用 `final_review.md` 做临考收束

## 说明

这套材料目前只围绕课堂重点线索中提到的内容展开，没有系统扩展到整门课全部章节。
